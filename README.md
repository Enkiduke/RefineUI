# RefineUI

RefineUI is a minimalist World of Warcraft UI addon for **Retail** that polishes the default Blizzard UI instead of replacing it.

## Philosophy

**Refine, don't replace.**

RefineUI focuses on improving the look, layout, and usability of Blizzard's existing frames while keeping the core UI familiar.

## What It Includes

- Default UI styling and skinning
- Unit frame improvements
- Nameplate improvements
- Action bar refinements
- Chat and tooltip polish
- Bags, maps, quest, and misc quality-of-life modules
- Native Edit Mode integration support

## Install

1. Download or clone this repository.
2. Place the `RefineUI` folder in:
   `World of Warcraft\\_retail_\\Interface\\AddOns\\`
3. Restart WoW (or reload UI).
